Chinese Word Segmentation Component

分词使用方法：


    List<Word> words = WordSeg.seg("杨尚川是APDPlat应用级产品开发平台的作者");
    System.out.println(words);
    
    输出：
    [杨尚川, 是, APDPlat, 应用, 级, 产品开发, 平台, 的, 作者]
